# Python Calculator

A simple Python calculator that performs basic arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Run

#### Windows Terminals
    python CLC.py

#### Bash / Linux Based
    python3 CLC.py
